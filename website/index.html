Quick Start

Get up and running in 3 steps

1

Clone & Configure

Clone the repository and set your target ESP32-S3 board

git clone https://github.com/beancookie/xiaoclaw.git
cd xiaoclaw
idf.py set-target esp32s3
2

Configure Secrets

Set up your WiFi and API keys in mimi_secrets.h

#define MIMI_SECRET_WIFI_SSID "YourWiFi"
#define MIMI_SECRET_WIFI_PASS "YourPassword"
#define MIMI_SECRET_API_KEY "sk-ant-..."
3

Build & Flash

Build the firmware and flash to your device

idf.py build
idf.py -p PORT flash monitor
